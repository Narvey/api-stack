This project runs 100% on Github, and you can access via its Github repository, with a machine readable APIs.json index available for all for the organizations, and APIs included in this stack. You can access all repositories using the master APIs.json collections for this work, with each individual service provider operating as its own Github repository.

It is important to remember that this is my workbench for profiling APIs, and it is all a work in progress. If you are integrating with, and consuming API definitions, it is up to you to build in healthy practices for keeping in sync, and complete within your own definition.
